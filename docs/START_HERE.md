# START HERE

This manual teaches you how to build **any new application** using your Antigravity ecosystem:

- **Workflows** (slash commands) define the **sequence** of steps.
- **Skills** define **how** to perform those steps with discipline.
- **Rules** (AGENTS.md, GEMINI.md) constrain behavior and reduce noise.
- **MCPs** add structured reasoning + accurate docs.
- **Graphify** provides a persistent architecture map to reduce re-reading and tokens.

## The one-sentence method

> **Clarify → Plan → Implement → Verify → Review → Document → Close Session → (Optional) Knowledge Item**

## Your minimal daily loop

1. **Frame the task** (2–6 sentences).
2. If unclear → **Brainstorm** (only when needed).
3. Run **`/plan`** and approve the architecture.
4. Implement using **`/tdd`** (behavior) or Fast Mode (trivial).
5. If something breaks → **`/debug`**.
6. Gate with **`/review`**.
7. Update docs if needed → **`/document`**.
8. End with **`/close-session`**.
