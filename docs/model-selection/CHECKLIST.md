# Model Selection Checklist

Ask these questions:

1. Is the task mechanical and well-defined?
   - Yes → Fast model

2. Does it span multiple files or require integration judgment?
   - Yes → Standard model

3. Is it architecture / security / deep debugging / high risk?
   - Yes → Most capable model

4. Have you failed 2–3 times with the current model?
   - Yes → escalate one tier
