# Optimization Protocol

1. Invoke the **performance-audit** skill on the target code
2. Generate the audit report with findings
3. Prioritize: Quick Wins first, then Planned Improvements
4. For each optimization:
   - Document the current behavior/metric (before)
   - Apply the optimization
   - Run tests — ensure no behavior change
   - Document the improved metric (after)
5. Summary report:

| # | Optimization | Before | After | Improvement |
|---|-------------|--------|-------|-------------|

6. Recommend any architectural changes for Phase 2 (if needed)
